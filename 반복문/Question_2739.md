## Question 2739


```python 3
N = int(input())

# N은 1보다 크거나 같고, 9보다 작거나 같다 (0<N<10)# 
for i in range(1,10):  # range() = <= i <, 주의하자! #
    print(N, '*', i, '=', N*i)
```
